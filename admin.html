Coordonnées bancaires (virements)

Affichées sur les factures et dans l'e-mail de paiement par virement.

🎁 Code de bienvenue à l'inscription

Quand c'est activé, chaque nouveau client reçoit automatiquement par e-mail, à sa première connexion, un code cadeau utilisable une seule fois, sans date d'expiration. Un seul code par compte.

Numérotation des factures

Numéro attribué à la prochaine facture payée (format AAAA-NNNN). À modifier avec prudence : choisir un numéro déjà utilisé créerait un doublon.

Facturation mensuelle partenaires

Génère un document unique regroupant les courses terminées d'un partenaire sur un mois (facturation centralisée). Active « Fact. mensuelle » sur le partenaire dans Clients pour qu'il apparaisse ici.
Gestion : les comptes clients se gèrent dans Clients (suppression) ; les rôles admin et les points dans Fidélité ; les codes promo et avoirs dans Codes promo.
Les modifications d'identité et de coordonnées bancaires ci-dessus s'appliquent immédiatement aux nouveaux devis, factures et e-mails de virement.
